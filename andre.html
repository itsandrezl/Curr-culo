Tutorial das tags do html

O que é HTML?

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Principais Tags HTML

  • html: Define o início e o fim do documento HTML.

  • head: Contém informações sobre o documento, como título, metadados e links para estilos.

  • body: Contém o conteúdo visível da página, como texto, imagens, vídeos, etc.

  • header: Define o cabeçalho da página ou de uma seção.

  • section;: Define uma seção dentro de um documento.

  • div: Define uma divisão ou seção genérica em um documento.

  • footer: Define o rodapé da página ou de uma seção.

  • h1 a h6: Define cabeçalhos de diferentes níveis de importância.

  • hr: Cria uma linha horizontal para separar conteúdos.

  • p: Define um parágrafo de texto.

  • ul: Define uma lista não ordenada.

  • li: Define um item de lista dentro de uma lista ordenada ou não ordenada.

  • br: Insere uma quebra de linha.

10 novas tags pra se usar no html!

  • a: Define um hiperlink para outra página.

  • img: Insere uma imagem na página.

  • input: Cria um campo de entrada, como um campo de texto ou botão.

  • table: Define uma tabela.

  • tr: Define uma linha em uma tabela.

  • td: Define uma célula em uma tabela.

  • form: Define um formulário para entrada de dados do usuário.

  • button: Cria um botão clicável.

  • span: Define uma pequena seção de texto.

  • iframe: Incorpora outro documento HTML dentro do atual.
